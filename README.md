 🎬 Catálogo de Filmes

Projeto full stack com **Node.js**, **Express**, **MongoDB** e **React** para cadastrar, editar e listar filmes com imagens.

---

📌 Tecnologias
- Node.js + Express
- MongoDB + Mongoose
- React + Vite
- Thunder Client (testes de API)

---

✨ Funcionalidades
- 🔍 Listar filmes
- ➕ Cadastrar novo filme
- ✏️ Atualizar filme
- ❌ Deletar filme por ID
- 🧹 Deletar todos os filmes
